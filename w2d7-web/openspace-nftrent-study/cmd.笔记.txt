
### 获取项目模板并启动项目
git clone https://github.com/0xqige/openspace-nftrent-study
cd openspace-nftrent-study/
npm install -g pnpm
cd web/
npm install
pnpm dev

# 浏览器访问 http://localhost:3001 观看效果.
